User Section

Anforderungen

  • Node.js Version 18.0 oder höher (prüfen mit node -v).
  • Für die Verwaltung mehrerer Node-Versionen: nvm verwenden.

Project Website

Der einfachste Weg, Docusaurus zu installieren, ist die Verwendung des create-docusaurus-Befehls:

npx create-docusaurus@latest my-website classic

Sie können auch TypeScript verwenden, indem Sie den --typescript-Schalter hinzufügen:

npx create-docusaurus@latest my-website classic --typescript

Meta-Only

Wenn Sie ein Docusaurus-Projekt für eine Meta-Open-Source-Seite einrichten, verwenden Sie diese spezielle Konfiguration.
